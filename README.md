# Backend

## Dependencies
Install dependencies with 
### `npm install`

## Testing
To run the full suite of test, you can run:
### `npm test`

To run api tests, you can run:
### `npm run test:api`

To run unit tests, you can run:
### `npm test:unit`

## Run locally
To start the backend server locally, run:
### `npm start`
